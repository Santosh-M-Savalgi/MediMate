# MediMate

## Overview

MediMate is a healthcare consultation platform designed to bridge the gap between patients and healthcare professionals through an online community-based system. The platform enables patients to share their medical concerns, receive guidance from verified doctors, and schedule consultations in a secure and user-friendly environment.

The project aims to provide accessible healthcare support by integrating social interaction with digital medical consultation services.

---

## Features

- User authentication and authorization
- Doctor and patient role management
- Health issue posting system
- Doctor comment and consultation system
- Doctor verification process
- Appointment booking and management
- Private messaging between patients and doctors
- Doctor rating and feedback system
- User profile management
- Responsive web interface

## Installation

### Clone the Repository

```bash
git clone https://github.com/Santosh-M-Savalgi/MediMate.git
```

Navigate to the project directory:

```bash
cd MediMate
```

---

## Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

## Running the Application

### Start the Backend Server

```bash
npm start
```

or

```bash
node server.js
```

### Start the Frontend Development Server

```bash
npm run dev
```

The application will be available on the configured local host.
